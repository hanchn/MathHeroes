# MathHeroes (数学小英雄)

## 1. 产品的功能
- **基础算术闯关**：支持自定义难度（10以内、20以内、100以内等）的加减法闯关游戏。
- **乘法口诀表**：提供交互式的乘法口诀学习与记忆测试。
- **数感培养模块**：包含“数字组成拆分”、“数位可视化（个十百千）”、“数字详细属性解析”等功能。
- **书写引导**：指导儿童在屏幕上正确书写数字。

## 2. 作用
- 寓教于乐，帮助学龄前及小学低年级儿童完成数学启蒙。
- 将抽象的数理逻辑（如进位、拆分）转化为直观的可视化游戏，培养孩子的核心数感。

## 3. 如果通过vibecoding去实现
- **视觉风格设计**：提示AI“生成一套适合儿童的、色彩丰富且带有卡通元素的 UI 组件”，AI会自动配置 Tailwind 颜色主题与圆角按钮。
- **算法生成**：让 AI 编写核心的 `mathGenerator.js`，只需描述规则（如“生成20以内的进位加法，避免重复”），AI即可产出完备的算术生成算法。
- **路由与状态流转**：要求 AI 配置 Vue Router，将首页、游戏页、结算页串联，并实现游戏积分与通关状态的全局管理。

## 4. 框架架构
- **前端框架**：Vue 3 + Vite
- **路由管理**：Vue Router
- **部署方案**：原生支持 Vercel 部署配置
